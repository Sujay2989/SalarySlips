Examples of Block and inline elements :


Block-level elements:
<div> - a generic container.


<div>
    This is a block-level element.
</div>
<p> - defines a paragraph.


<p>
    This is a block-level paragraph.
</p>
<h1> to <h6> - heading elements.


<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<!-- ... -->
<h6>This is a Heading 6</h6>
<ul> and <li> - unordered list and list items.


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
<ol> and <li> - ordered list and list items.


<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>
Inline elements:
<span> - a generic inline container.


<p>
    This is an <span>inline</span> element.
</p>
<strong> - represents strong importance or importance of its contents.


<p>
    This is a <strong>strong</strong> element.
</p>
<em> - represents emphasized text.


<p>
    This is <em>emphasized</em> text.
</p>
<a> - creates hyperlinks.


<p>
    Visit our <a href="https://www.example.com">website</a>.
</p>
<img> - embeds images.


<p>
    Here is an <img src="image.jpg" alt="Description"> inline image.
</p>

Example of Attribute Selector:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    /* Select all elements with the "data-highlight" attribute */
    [data-highlight] {
      color: red;
      font-weight: bold;
    }

    /* Select all input elements with the "required" attribute */
    input[required] {
      border: 2px solid #007bff;
    }

    /* Select all links with an href attribute containing "example.com" */
    a[href*="example.com"] {
      color: green;
    }
  </style>
  <title>Attribute Selector Example</title>
</head>
<body>

  <h1 data-highlight>This heading is highlighted</h1>

  <p>This paragraph is not highlighted</p>

  <form>
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>

    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
  </form>

  <p><a href="https://www.example.com">Visit Example.com</a></p>

  <p><a href="https://www.anotherwebsite.com">Visit Another Website</a></p>

</body>
</html>


The first rule selects all elements with the data-highlight attribute and applies a red color and bold font to them.
The second rule selects all input elements with the required attribute and adds a blue border to them.
The third rule selects all a (link) elements with an href attribute containing "example.com" and changes their color to green.

Examples of Borders:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="styles.css">
    <title>Bordered Element</title>

body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
    background-color: #f0f0f0;
}

.bordered-element {
    border: 2px solid #333;
    padding: 20px;
    border-radius: 8px;
    background-color: #fff;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
}

In the HTML file, we have a simple structure with a div element having a class of "bordered-element" and containing a paragraph (<p>) with some content.

In the CSS file, we style the body to center the content on the page and set a background color. The ".bordered-element" class defines the styles for the bordered element.

border: Specifies the border of the element. In this case, it's a 2px solid border with the color #333.
padding: Adds some padding inside the element.
border-radius: Adds rounded corners to the border.
background-color: Sets the background color of the element.
box-shadow: Adds a subtle box shadow for a sense of elevation.

Example of Media Queries :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Layout</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        header {
            background-color: #333;
            color: #fff;
            text-align: center;
            padding: 10px;
        }

        main {
            padding: 20px;
        }

        footer {
            background-color: #333;
            color: #fff;
            text-align: center;
            padding: 10px;
            position: fixed;
            bottom: 0;
            width: 100%;
        }

        /* Media query for devices with a maximum width of 600px */
        @media (max-width: 600px) {
            main {
                padding: 10px;
            }
        }

        /* Media query for devices with a minimum width of 601px and a maximum width of 900px */
        @media (min-width: 601px) and (max-width: 900px) {
            main {
                padding: 15px;
            }
        }
    </style>
</head>
<body>

    <header>
        <h1>Responsive Layout Example</h1>
    </header>

    <main>
        <p>This is the main content of the page. The padding of this section will change based on the screen width.</p>
    </main>

    <footer>
        <p>© 2024 Your Website</p>
    </footer>

</body>
</html>


The layout consists of a header, main content area, and a footer.
The @media queries are used to define different styles for different screen widths.
The default styles apply to screens wider than 900px.
For screens with a width between 601px and 900px, the padding of the main element is adjusted.
For screens with a width of 600px or less, the padding of the main element is further reduced for better mobile display.
